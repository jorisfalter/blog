Introduction

Link Summary ("the Extension") is designed to enhance your browsing experience by allowing you to generate summaries of articles and webpage content through the OpenAI API. We are committed to protecting your privacy and ensuring transparency about how we handle your data.

What Data Do We Collect?

The Extension does not collect, store, or share any personal data. However, it may interact with external services and APIs as part of its functionality.

  • API Key Storage: The Extension uses Chrome's storage.local feature to securely store your OpenAI API key. This key is provided by you and is only accessible within the Extension to facilitate API requests.
  • URL Processing: The Extension processes the URL of the webpage or link you want to summarize. These URLs are temporarily sent to the OpenAI API to generate summaries. The URLs are not stored or shared by the Extension.

How Is Your Data Used?

The Extension uses the data you provide solely for its intended purpose:

  • API Key: The OpenAI API key is used to authenticate requests to OpenAI's servers. The key is not shared outside of the Extension or stored on any external server.
  • Webpage URLs: URLs are sent to OpenAI's API to generate content summaries. These URLs are not logged or stored by the Extension after the API call.

Do We Share Your Data?

We do not sell, share, or disclose your data to any third parties. However, data such as webpage URLs are sent to the OpenAI API for processing, as this is necessary for the functionality of the Extension. Please refer to OpenAI's Privacy Policy for details on how they handle your data.

How Is Your Data Secured?

  • Local Storage: The OpenAI API key is securely stored using Chrome's storage.local API. This ensures that the key is isolated to your browser and is not accessible by other websites or extensions.
  • Secure API Requests: All requests to the OpenAI API are encrypted using HTTPS to ensure the security of your data in transit.

User Responsibility

To ensure the safe usage of the Extension:

  • Do not share your OpenAI API key with others.
  • Use the Extension responsibly and avoid summarizing sensitive or private content.

Your Choices

You can manage your OpenAI API key through the Extension's settings. At any time, you can:

  • Remove the stored API key via the Chrome extension settings or options page.
  • Uninstall the Extension to remove all stored data.

Changes to This Privacy Policy

We may update this Privacy Policy from time to time to reflect changes in our practices or for other operational, legal, or regulatory reasons. When changes occur, the updated Privacy Policy will be made available via the Extension’s listing in the Chrome Web Store.
